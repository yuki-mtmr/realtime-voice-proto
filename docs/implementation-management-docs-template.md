## 実装ルール

1. Figmaが唯一の正（Single Source of Truth）
2. 実装前に必ず該当セクションのFigmaデータをMCPで取得する
3. 1セクションずつ実装する。複数セクションを同時に扱わない
4. このドキュメントとFigmaが矛盾する場合はFigmaを優先し、このドキュメントを更新する
5. 実装完了後にこのドキュメントのステータスを更新する
